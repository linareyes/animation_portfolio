About Project

This project was meant to incorporate animations into a second personal portfolio in order to create a cohesive user-friendly website. The animations reinforce the content I provided on the website by using hierarchy, relationships, structure, causes and effect. The main focus in this website was making my logo and information memorable. The first step in accomplishing this was allowing my audience know and see the symbol/logo that represents me. I was able to accomplish this by creating a logo that draws itself while making it extremely big and bold to make it pop out. Secondly I created an "explore" button in relationship to the end product of my logo. It only appears when my logo is done drawing itself. Which then creates a cause and effect by giving the user the options of clicking in order to explore more content. Once clicked the website transitions into my landing page which then explains who I am and what I do. I created an interactive friendly website in order to allow users to see what kind of work I do.
